Multiple\Backend\Controllers\IndexController::indexAction

Verifica para qual pagina deve ser redirecionado o acesso de acordo com os dados que estão faltando no banco de dados.

Caso ainda não exista o arquivo de conexão com o banco de dados, redireciona para o formulário para criar o arquivo com os dados recebidos. Caso não exista nenhum usuário redireciona para tela de criação de usuários. Caso ambos estejam OK, redireciona para tela de login.
